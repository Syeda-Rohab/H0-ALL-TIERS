## 1️⃣ BRONZE TIER (Basic)

**Location:** `bronze\` folder

### Run Option A: Double-click
1. Go to `bronze` folder
2. Double-click `run.bat`

### Run Option B: Command Line
```bash
cd bronze
python ai_employee_simple.py
```

### What It Does:
- ✅ Simulates receiving 3 emails
- ✅ Creates EMAIL_*.md files in Needs_Action
- ✅ Creates PLAN_*.md files (action plans)
- ✅ Moves processed files to Done
- ✅ Updates Dashboard.md

### Output:
Check `bronze\bronze_vault\Done\` folder for processed files!

## 2️⃣ PLATINUM TIER (Advanced)

## 3️⃣ UI TIER (Web Dashboard)

**Location:** `UI-TIER\` folder

### First Time Setup:
```bash
cd UI-TIER
npm install
```

### Run:
```bash
npm run dev
```

Then open: **http://localhost:3000**

### What It Does:
- ✅ Beautiful web dashboard
- ✅ Real-time statistics
- ✅ Email management interface
- ✅ Task tracking
- ✅ Activity feed
- ✅ Can deploy to Vercel
